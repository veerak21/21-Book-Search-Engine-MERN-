# Project Title: Book Search Engine Starter Code

## Description:
 This is a Google Books API search engine for new books for user to read.


## Installation:
  - Node.js
  - Express
  - Mongodb
  - React
  - GraphQl
  - Apollo-server-express
  - Apollo-client
  - JWT(jsonwebtoken)
